# Stock Portfolio Management System

## Project Overview
This project helps users track stock investments and calculate portfolio values efficiently. Users can enter stock symbols and quantities, and the system generates investment summaries and reports automatically.

## Features
- Portfolio Value Calculation
- Multiple Stock Support
- Investment Tracking
- Report Generation
- File Handling
- Portfolio Summary

## Technologies Used
- Python
- Dictionaries
- Functions
- File Handling
- User Input Processing

## Project File
- Stock_Portfolio_Manager.py
